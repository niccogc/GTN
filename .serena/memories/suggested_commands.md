# Suggested Commands

## Testing
- Run tests: `python test_*.py` (individual test files)
- Test environment caching: `python test_environment_caching.py`

## Development
- Python interpreter: `python` or `python3`
- Git operations: `git status`, `git add`, `git commit`, etc.

## System Commands (Linux)
- List files: `ls`
- Change directory: `cd`
- Search files: `find`
- Search content: `grep`
