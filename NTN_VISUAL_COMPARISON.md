# NTN Optimization: Visual Comparison

## 1. Ridge/Jitter Application Flow

### GTN: Unscaled Ridge
```
┌─────────────────────────────────────────────────────────────┐
│ 1. Compute Hessian H and Gradient b                         │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. Compute scale = mean(|diag(H)|)                          │
│    ⚠️  COMPUTED BUT NOT USED!                               │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. Apply Ridge (UNSCALED)                                   │
│    H' = H + 2*jitter*I                                      │
│    b' = b + 2*jitter*w_old                                  │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. Solve: (H + 2*jitter*I) Δw = -b'                        │
└─────────────────────────────────────────────────────────────┘
```

### TNOld: Scaled Ridge
```
┌─────────────────────────────────────────────────────────────┐
│ 1. Compute Hessian A and Gradient b                         │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 2. Compute scale = mean(|diag(A)|)                          │
│    ✓ COMPUTED AND USED!                                     │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 3. Normalize System (SCALED)                                │
│    A' = A / scale                                           │
│    b' = b / scale                                           │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 4. Apply Ridge to Scaled System                             │
│    A'' = A' + 2*eps*I                                       │
│    b'' = b' + 2*eps*w_old                                   │
└────────────────────┬────────────────────────────────────────┘
                     │
                     ▼
┌─────────────────────────────────────────────────────────────┐
│ 5. Solve: (A/scale + 2*eps*I) Δw = -b''                    │
└─────────────────────────────────────────────────────────────┘
```

## 2. Sweep Order Comparison

### GTN: Asymmetric Sweep (4 nodes)
```
EPOCH 1:
┌─────────────────────────────────────────────────────────────┐
│ FORWARD PASS                                                │
│ ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐                     │
│ │Node0 │→ │Node1 │→ │Node2 │→ │Node3 │                     │
│ └──────┘  └──────┘  └──────┘  └──────┘                     │
│    ↓        ↓        ↓        ↓                             │
│   [1]      [1]      [1]      [1]                            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ BACKWARD PASS (excludes first and last)                     │
│                ┌──────┐  ┌──────┐                           │
│                │Node2 │← │Node1 │                           │
│                └──────┘  └──────┘                           │
│                   ↓        ↓                                │
│                  [2]      [2]                               │
└─────────────────────────────────────────────────────────────┘

TOTAL UPDATES:
  Node0: 1x (forward only)
  Node1: 2x (forward + backward)
  Node2: 2x (forward + backward)
  Node3: 1x (forward only)
  
  Total: 6 updates per epoch
```

### TNOld: Symmetric Sweep (4 nodes)
```
SWEEP 1:
┌─────────────────────────────────────────────────────────────┐
│ LEFT-TO-RIGHT PASS                                          │
│ ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐                     │
│ │Node0 │→ │Node1 │→ │Node2 │→ │Node3 │                     │
│ └──────┘  └──────┘  └──────┘  └──────┘                     │
│    ↓        ↓        ↓        ↓                             │
│   [1]      [1]      [1]      [1]                            │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ RIGHT-TO-LEFT PASS                                          │
│ ┌──────┐  ┌──────┐  ┌──────┐  ┌──────┐                     │
│ │Node3 │← │Node2 │← │Node1 │← │Node0 │                     │
│ └──────┘  └──────┘  └──────┘  └──────┘                     │
│    ↓        ↓        ↓        ↓                             │
│   [2]      [2]      [2]      [2]                            │
└─────────────────────────────────────────────────────────────┘

TOTAL UPDATES:
  Node0: 2x (L2R + R2L)
  Node1: 2x (L2R + R2L)
  Node2: 2x (L2R + R2L)
  Node3: 2x (L2R + R2L)
  
  Total: 8 updates per epoch
```

## 3. Regularization Strength Comparison

### Fixed vs Adaptive Regularization

```
Scenario: Different Hessian Magnitudes
jitter/eps = 1e-6

┌──────────────────────────────────────────────────────────────┐
│ CASE 1: Large Hessian (scale = 1000)                        │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ GTN (Fixed):                                                │
│   λ_eff = 2 * 1e-6 = 2e-6                                   │
│   ┌─────────────────────────────────────────────────────┐   │
│   │ Regularization: STRONG (relative to small H)        │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
│ TNOld (Adaptive):                                           │
│   λ_eff = 2 * 1e-6 / 1000 = 2e-9                           │
│   ┌─────────────────────────────────────────────────────┐   │
│   │ Regularization: WEAK (relative to large H)          │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ CASE 2: Small Hessian (scale = 0.001)                       │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ GTN (Fixed):                                                │
│   λ_eff = 2 * 1e-6 = 2e-6                                   │
│   ┌─────────────────────────────────────────────────────┐   │
│   │ Regularization: WEAK (relative to large H)          │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
│ TNOld (Adaptive):                                           │
│   λ_eff = 2 * 1e-6 / 0.001 = 2e-3                          │
│   ┌─────────────────────────────────────────────────────┐   │
│   │ Regularization: STRONG (relative to small H)        │   │
│   └─────────────────────────────────────────────────────┘   │
│                                                              │
└──────────────────────────────────────────────────────────────┘

KEY INSIGHT:
  GTN:    "Always regularize by the same amount"
  TNOld:  "Regularize proportionally to Hessian magnitude"
```

## 4. Numerical Stability Comparison

### Ill-Conditioned Hessian Example

```
H = [1000    0  ]    scale = 500
    [   0    0.1]

┌──────────────────────────────────────────────────────────────┐
│ GTN: UNSCALED SYSTEM                                         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ H + 2λI = [1000+2λ    0      ]                              │
│           [   0    0.1+2λ   ]                               │
│                                                              │
│ If λ=1e-6:                                                  │
│ [1000.000002    0        ]                                  │
│ [   0         0.100002  ]                                   │
│                                                              │
│ Condition number ≈ 10,000 (STILL ILL-CONDITIONED!)         │
│ ⚠️  Regularization barely helps!                            │
│                                                              │
└──────────────────────────────────────────────────────────────┘

┌──────────────────────────────────────────────────────────────┐
│ TNOld: SCALED SYSTEM                                         │
├──────────────────────────────────────────────────────────────┤
│                                                              │
│ (H/500 + 2ε*I) = [2+2ε    0    ]                            │
│                  [  0   0.0002+2ε]                          │
│                                                              │
│ If ε=1e-6:                                                  │
│ [2.000002    0      ]                                       │
│ [  0      0.0002002]                                        │
│                                                              │
│ Condition number ≈ 10,000 (SAME, but better numerical)     │
│ ✓ Scaling improves numerical behavior!                      │
│                                                              │
└──────────────────────────────────────────────────────────────┘
```

## 6. Memory Management Comparison

### GPU Memory Usage Over Time

```
GTN: Explicit Cleanup
┌─────────────────────────────────────────────────────────────┐
│ Memory                                                      │
│   ▲                                                         │
│   │  ┌─────────┐                                            │
│   │  │ Compute │                                            │
│   │  │ H, b    │                                            │
│   │  └────┬────┘                                            │
│   │       │ ┌──────────────────┐                            │
│   │       │ │ Solve system     │                            │
│   │       │ └────┬─────────────┘                            │
│   │       │      │ ┌──────────────────────┐                 │
│   │       │      │ │ Delete + empty_cache │                 │
│   │       │      │ └────┬─────────────────┘                 │
│   │       │      │      │                                   │
│   │ ──────┴──────┴──────┴──────────────────► Time           │
│   │                                                         │
│ ✓ Memory freed aggressively                                │
│ ✓ Better for memory-constrained GPUs                       │
└─────────────────────────────────────────────────────────────┘

TNOld: Implicit Cleanup
┌─────────────────────────────────────────────────────────────┐
│ Memory                                                      │
│   ▲                                                         │
│   │  ┌─────────┐                                            │
│   │  │ Compute │                                            │
│   │  │ A, b    │                                            │
│   │  └────┬────┘                                            │
│   │       │ ┌──────────────────┐                            │
│   │       │ │ Solve system     │                            │
│   │       │ └────┬─────────────┘                            │
│   │       │      │ (waiting for GC)                         │
│   │       │      │                                          │
│   │ ──────┴──────┴──────────────────► Time                  │
│   │                                                         │
│ ✗ Memory freed by garbage collection (slower)              │
│ ✗ May accumulate during long training                      │
└─────────────────────────────────────────────────────────────┘
```
